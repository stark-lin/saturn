Control References

按钮 / 小控件的几种候选形态。

Inset Button

最适合主按钮。像控制台内嵌按键，有实体感，但不靠外投影。

Flat Instrument

最适合次级操作。更像仪器面板上的普通控件，稳定、低噪音。

Chip Action

适合小操作、筛选器、局部工具栏。不要承担大 CTA。

Status Toggle

适合状态切换和系统标识，例如 local、indexed、LLM ready。

Online Indexing Offline

Slider / Switch

适合连续参数和二元开关，例如索引强度、自动同步、本地模式。
